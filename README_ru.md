## Особенности
+ Интеграция с [SAMPCTL](https://github.com/Southclaws/sampctl)
+ Разработан специально для [open.mp](https://github.com/openmultiplayer) & SA-MP
+ Менеджер зависимостей для серверных компонентов и пакетов
+ Подсветка цветов в формате RGBA/HEX прямо в редакторе кода
+ Встроенная поддержка Git с панелью управления версиями
+ Разделенный вид редактора для одновременной работы с несколькими файлами
+ Дерево проекта, оптимизированное для крупных проектов
+ Встроенные панели сборки и вывода данных сервера

## Установка
### Скачивание
1. Скачайте последнюю версию на странице 'Release'.
2. Распакуйте архив в любую папку.
3. Запустите 'Spawn.exe'.

### Дополнительные инструменты
Чтобы воспользоваться всеми функциями Spawn, вы также можете установить:
+ Git
+ [SAMPCTL](https://github.com/Southclaws/sampctl)

Эти инструменты можно настроить позже в настройках.

## Быстрый старт
### Создание проекта
1. Выберите 'Файл' → 'Новый проект...'.
2. Введите название проекта и выберите его местоположение.
3. После нажатия кнопки «Создать» запустится инструмент командной строки SAMPCTL. Для завершения создания проекта следуйте инструкциям на экране.
4. По завершении процесса закройте утилиту командной строки SAMPCTL. Spawn автоматически откроет новый проект.

### Управление зависимостями
1. Выберите 'Проект' → 'Менеджер зависимостей'.
2. Введите имя пакета в поле 'Зависимость'.
3. Нажмите 'Установить'.

+ Чтобы удалить зависимость, выберите нужную из списка и нажмите 'Удалить'.
+ Чтобы проверить что все зависимости в порядке, нажмите кнопку под названием "Проверить'.

### Использование Git
1. Щелкните правой кнопкой мыши по корневой папке в дереве проекта → выберите 'Инициализировать репозиторий'. (Если репозиторий еще не был на сервере)
2. Создайте свой первый коммит.
3. Отслеживайте изменения прямо из Spawn.

Если репозиторий уже находится на сервере, Spawn автоматически его обнаружит (при условии, что в настройках включена поддержка Git)

### Сборка и запуск сервера
1. Откройте проект.
2. Соберите ваш сервер.
3. Запустите ваш сервер.
4. Следите за выводом данных во встроеной консоли.

Вы готовы к разработке серверов open.mp и SA-MP с помощью Spawn.

## Разработка
### Требования
+ Python 3.13+
+ wxPython
+ GitPython
+ Markdown
+ watchdog

### Установка зависимостей
```python
pip install -r requirements.txt
```
### Запуск Spawn
```python
python main.py
```

## Стабильность
Spawn включает автоматизированные тесты для:
- Git интеграции
- Дерева проекта
- Системы конфигурации
- Управления зависимостями
- Загрузка файлов и определение кодировки

Все тесты запускаются автоматически при каждой фиксации с помощью GitHub Actions.
